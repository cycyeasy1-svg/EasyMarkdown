# Privacy Policy

Effective date: 2026-07-07

EasyMarkdown is a local-first Markdown editor. It does not require an account,
does not include built-in telemetry, and does not upload your documents by
default.

## Data Stored Locally

EasyMarkdown stores editor preferences, session state, recent files, custom
themes, and unsaved scratch tabs on your device. Markdown files and pasted
assets are read from or written to locations you choose.

## Network Access

EasyMarkdown may access the network only for product features that need it:

- update checks against the GitHub Releases API;
- links you open from the app;
- optional user-configured image upload commands or integrations.

No document content is sent by the built-in update check.

## Mobile File Access

On mobile builds, EasyMarkdown uses platform file APIs to open, import, save,
or share files that you choose. App-private storage remains on the device unless
you export or share a file.

## Security Reports

Please report security issues privately through the process in `SECURITY.md`.
