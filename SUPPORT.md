# Support

Use GitHub Issues for public bug reports and feature requests:

https://github.com/cycyeasy1-svg/EasyMarkdown/issues

For bug reports, include:

- EasyMarkdown version;
- operating system and version;
- steps to reproduce;
- expected result and actual result;
- screenshots, logs, or a small sample Markdown file when useful.

For security issues, do not open a public issue. Follow `SECURITY.md` instead.
