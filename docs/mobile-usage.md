# HorseMD 移动端使用说明(Android / iOS)

面向手机/平板用户的使用指南。技术实现见 [mobile.md](./mobile.md)。

> 移动端复用桌面同一套编辑器(React + Milkdown),做了触摸适配。桌面端不受影响。

## 一、安装

### Android
- 拿到 `HorseMD-x.x.x-release.apk`(GitHub Releases / 作者分享)。
- 传到手机(微信文件传输助手 / 网盘 / 数据线),点开安装;首次会提示「未知来源」,允许即可。
- 安装包未做花钱签名公证,但已用开发者证书签名,正常可用。

### iOS
- 目前需用 Xcode 安装到自己的设备(免费 Apple ID 即可,App 7 天后需重连 Xcode 续期);
  正式分发将走 TestFlight / App Store(需 Apple 开发者账号)。详见 [mobile.md](./mobile.md) §10。

## 二、界面速览

```
┌───────────────────────────────────────────┐
│ [系统状态栏:时间/电量,与顶栏同色融合]        │
│ ☰   标签1 ✕  标签2 ✕ …          +    ⌘     │  顶栏
├───────────────────────────────────────────┤
│                                           │
│            正文编辑区(所见即所得)           │
│         (大屏自动居中为舒适阅读宽度)         │
│                                           │
├───────────────────────────────────────────┤
│ ✓ 12 字 · 80 字符 · 1 分钟       导出   ⋯ 更多 │  底栏
└───────────────────────────────────────────┘
```

- **☰**:打开左侧抽屉(主页 / 文件树 / 大纲)。
- **标签**:多个文档同窗;点标签切换,点标签上的 **✕** 关闭。
- **+**:新建文档。
- **⌘**:命令面板(搜索文件 / 执行命令)。
- **底栏左**:保存状态(✓ 已保存 / ● 已修改)+ 字数/字符/阅读时长。
- **底栏右**:**导出**(系统分享)、**⋯ 更多**(保存 / 源码 / 主题 / 语言 / GitHub)。

## 三、基本操作

### 新建 / 打开
- **新建**:顶栏 `+`,或首页「新建文件」,或命令面板「新建文件」。
- **打开文件**:首页「打开文件」/ 命令面板 → 走系统文件选择器,选中的 `.md` 会拷进 App 本地库并打开。
- **打开文件夹(= App 本地库)**:☰ → 文件图标 →「打开文件夹」,列出你保存过的所有文档。
- **用其他 App 打开 .md**:在「文件」App 或别处分享 `.md` → 「用 HorseMD 打开 / 拷贝到 HorseMD」,会自动导入并打开。

### 写作
- 直接输入即所见即所得:`# ` 标题、`- ` 列表、`> ` 引用、` ``` ` 代码块、表格、任务列表、`$$` 数学公式、` ```mermaid ` 图表等。
- 切到**源码模式**:⋯ 更多 → 源码(再点回到富文本)。

### 保存
- **⋯ 更多 → 保存**;未命名文档会**弹窗让你填文件名**。
- 保存后底部提示「已保存 ✓ / 文件名 / 位置」,约 5 秒自动消失(也可点 ✕ 关闭)。
- **保存位置**:
  - **iOS**:系统「文件」App → 我的 iPhone → HorseMD(可在文件 App 里直接找到)。
  - **Android**:App 本地库(`Android/data/.../HorseMD/`)。系统「文件」App 看不到该目录(系统限制),但 **App 内随时可见**(☰ → 打开文件夹),要拿到外面用下面的「导出/分享」。

### 导出 / 分享
- 底栏 **导出** 按钮(或对未保存文档会提示先保存)→ 弹系统分享面板 → 可「**存储到文件**」、发到网盘 / 邮件 / 聊天、AirDrop(iOS)等。
- 这是把文档复制到 App 外、你能找到的位置的推荐方式(Android 尤其)。

### 抽屉(☰)
- 顶部 ☰ 打开;包含**主页**(头像,点了直接回欢迎页并收起抽屉)、**文件树**、**大纲**。
- 点文件树里的文件、或大纲里的标题 → **打开/跳转后抽屉自动收起**,直接看到内容。
- 点遮罩或抽屉底部的收起按钮可关闭。

### 主题 / 语言 / 查找
- **主题**:⋯ 更多 → 主题色板(暖光 / 暖夜 / 四套莫兰迪)。
- **语言**:⋯ 更多 → English / 中文。
- **查找**:命令面板 →「查找」。

## 四、手机端不提供的功能(桌面端有)

这些在手机上不适用,已隐藏;桌面端不变:

- 导出 PDF、分屏、在访达/资源管理器中显示、窗口控件、图床自定义命令。

## 五、小贴士

- 文档**全宽自适应**手机屏;平板/横屏会居中为约 760px 阅读宽度。
- 双击空白不会再放大页面(已锁定缩放);只上下滚动,不会左右乱滑。
- 找不到保存的文件?**Android 在 App 内「打开文件夹」里看,或用「导出」发出去**;iOS 在系统「文件」App 里。
