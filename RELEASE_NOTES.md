## v1.2.1
<!-- head: 5d779a4 -->

EasyMarkdown v1.2.1 更新日志
本次更新修复了保持模式下的大纲、表格与查找问题，并改善了标签页和复制操作的可靠性。

问题修复
- 修复保持模式下滚动文档时，大纲无法正确跟随当前标题，以及点击大纲标题可能无法准确跳转的问题
- 修复保持模式下使用键盘切换表格单元格时，目标单元格可能被固定表头遮挡的问题
- 修复保持模式下表格固定表头无法点击选中单元格、无法打开右键菜单，以及选中状态不同步的问题
- 修复表格筛选后，文内查找仍会统计已隐藏行，导致上一个或下一个结果无法定位的问题
- 修复正式安装版中复制文件路径或文件名可能没有写入剪贴板的问题
- 修复标签页较多时宽度不均、文件名末尾可能被关闭按钮遮挡的问题

## v1.2.0
<!-- head: fcc772d -->

EasyMarkdown v1.2.0 更新日志
本次更新聚焦于更安全的保持模式编辑、更高效的文档导航与更完整的内置帮助：加入修改审阅、本地历史、链接管理、源码同步分栏和标签恢复等能力，并显著改善大型表格与长文档体验。

新增与优化
- 新增 F1 帮助中心：提供中、英、日三语使用指南与全文搜索，可从欢迎页、状态栏、设置、菜单和命令面板打开
- 保持模式新增独立的撤销与重做；未确认的单元格或内容块草稿会跟随标签保留，保存、切换模式或撤销前会要求确认处理
- 新增「查看本次修改」：按源码范围审阅当前内容与上次保存版本的差异，可定位或单独恢复某项修改；遇到外部文件冲突时也可先比较
- 新增可选的持久化本地历史：保存前的旧版本仅保留在当前设备，可比较、恢复或删除；每个文档最多 30 版、保留 30 天
- 保持模式新增「富文本 + 源码」同步分栏，可交换左右位置、固定预览并启用双向滚动同步
- 标签页新增预览标签、最近使用标签切换器和关闭标签恢复；`Ctrl/Cmd+Shift+T` 现在用于恢复最近关闭的文件
- 新增统一的跳转历史：通过链接、大纲、搜索结果或行号跳转后，可用 `Alt/Option+←/→` 或鼠标侧键返回和前进
- 增强 Markdown 内部链接：支持应用内跳转、右侧分屏打开、问题诊断、引用查询，以及在重命名标题、文件或移动文档时预览并更新相关链接
- 命令面板新增搜索范围下拉框，可分别查找文件、命令、当前文档标题、工作区标题、行号和帮助；`>`、`@`、`#`、`:`、`?` 快速前缀继续保留
- 文件树支持完整键盘操作、剪切移动和隐藏文件显示设置；大型工作区的展开与搜索更清晰
- 保持模式表格支持键盘选中与移动、Excel/TSV 矩形粘贴、行列操作和快捷筛选；筛选候选值现在显示出现次数
- 大型表格改为分批渲染，打开、滚动、搜索和编辑时更加流畅
- 源码视图按标签记忆状态，切换视图时保留光标与滚动位置；搜索高亮、折叠行定位和输入中的行内公式预览更加准确
- 图片、Mermaid 图和公式支持统一的放大、缩放与拖动查看；富文本斜杠菜单支持用拼音搜索，代码块内按 Tab 可直接插入缩进
- 新增「添加附件」命令：将文件复制到文档旁的资源目录并插入相对链接
- 可分别设置英文、中文、日文和代码字体，导出 PDF、HTML 与打印时同样生效；字号、缩放、行间距等高频排版项集中到状态栏
- 新增 Zen 专注模式（`Ctrl/Cmd+K` 后按 `Z`），可临时隐藏应用界面并在靠近屏幕边缘时唤回
- 保持模式支持安全显示空的 HTML 文档锚点，并可正确跳转到折叠内容中的锚点
- 外部链接仅允许安全的网页、邮件协议，避免 Markdown 中的未知协议被系统直接打开

问题修复
- 修复富文本段落含转义下划线时，按 Enter 可能造成文字缺失、错误斜体或无法分段的问题
- 修复查找面板打开期间编辑正文后，高亮和结果数量不会及时更新的问题
- 修复大纲跳转在文档分批渲染或布局变化时偶尔偏离目标标题的问题
- 修复包含链接的表格按隐藏 URL 而非实际显示文字计算列宽，导致列宽异常的问题
- 修复短内容表格的表头可能被筛选与列控制按钮挤压截断的问题
- 修复网址后紧跟中文、日文等字符时，自动链接可能错误吞入后续文字的问题
- 修复连续粘贴同名图片时可能发生文件名冲突的问题

## v1.1.0
<!-- head: a10639f -->

EasyMarkdown v1.1.0 更新日志
本次更新幅度较大：补齐了查找替换、工作区全文搜索、打印、HTML 导出与统一设置面板等一批常用功能，保持模式的表格和任务列表更好用，日语文档的汉字终于按日文字形显示，同时明显改善了大文档编辑与启动时的流畅度。

新增与优化
- 工作区全文搜索（Ctrl+Shift+F）：在已打开的文件夹内跨文件搜索，支持区分大小写、全词匹配与正则，点击结果直接跳到对应行
- 查找替换（Ctrl+H，macOS 为 ⌥⌘F）：支持替换单个与全部替换，保持模式、富文本、源码三种视图下结果一致
- 统一设置面板（Ctrl+,）：编辑、排版、外观、语言集中到一处；状态栏原有的主题、语言、页面宽度等入口一并收进面板
- 打印（Ctrl+Alt+P）：直接调用系统打印
- 导出为 HTML（Ctrl+Shift+H）：单文件自包含，图片一并内嵌，发给别人可直接用浏览器打开
- 自动保存（默认关闭）：开启后停止输入约 2 秒自动写盘，未命名文档不受影响
- 标签页可拖拽排序，也可固定；固定的标签靠左排列，且不会被「关闭其他标签」误关
- 最近打开的文件支持固定、单条删除与整体清空
- 可将 EasyMarkdown 设为 .md / .markdown 等文件的默认打开程序
- 新增拼写检查（默认关闭）：开启后显示红色波浪线，右键给出修改建议
- 可指定新建标签默认使用保持模式还是富文本模式，单个标签仍可随时切换
- 应用菜单跟随界面语言（中 / 英 / 日）
- 打开文件时侧边栏自动切换：文件在工作区内切到「文件」，在工作区外切到「大纲」
- 未保存内容切换「保持 / Milkdown」模式时，新增专用确认窗口，可选择保存后切换、直接切换或取消，避免意外产生格式差分
- 文内查找会从当前光标或可见位置开始定位，长文档中查找下一处结果更自然
- 保持模式支持 GFM 任务列表（可直接勾选）与 YAML frontmatter（显示为信息卡片）
- 保持模式的行内 Markdown 渲染更接近富文本模式：删除线、下划线强调、自动链接、带标题的链接、包含括号的 URL、反斜杠转义以及高亮标记都能更准确显示
- 保持模式会按宽松列表显示被空行隔开的列表项，阅读节奏更接近常见 Markdown 预览
- 新增「保留连续空行」设置：保持模式可把连续空行显示为额外留白；只影响显示，不会改动文件内容，PDF / HTML 导出也会沿用
- 保持模式表格：可一键清除单个表格或整篇文档的筛选；文档内有多个表格时，状态栏显示「N 个表 · 筛选 x/y」并在悬停时列出各表明细
- 保持模式表格的列宽计算更合理
- 保持模式表格支持拖动调整列宽、双击自动适配、隐藏与恢复列；筛选或内容刷新后仍会保留当前显示状态
- 日语文档字体：含假名的文档自动切换到日文字体，其中的汉字按日文字形显示；导出 PDF / HTML 与打印同样生效。日文字体顺序调整为 Noto Sans JP → BIZ UDPGothic → Hiragino → Yu Gothic Medium → Meiryo
- 保存成功后弹出「已保存 ✓」提示；各类提示按成功 / 失败 / 进行中区分图标与颜色
- 折叠着的文件夹可右键「刷新」重新扫描；命令面板每次打开都会重建文件索引，新建的文件立刻能搜到
- 大文档更流畅：含长代码块的文档不再被误判为「重文档」而降级显示；源码模式下逐字输入不再随文件增大而变卡
- 含大量流程图的文档，快速滚动时渲染更稳定
- 启动更快：启动时的更新检查改到界面就绪之后进行，不再与恢复标签页、加载首个文档争抢资源
- 输入、切换标签、命令面板等日常操作的重绘开销降低，整体更跟手

问题修复
- 修复浅色主题下启动瞬间闪过一片暗色的问题
- 修复启动时窗口先以还原尺寸画一帧、再最大化，导致边框闪烁的问题
- 修复冷启动时双击 .md 文件、或用「用 EasyMarkdown 打开」文件夹，偶尔内容不加载的问题；首屏被杀毒软件拖慢时，窗口最迟约 3 秒也会显示，不会看起来像没启动
- 修复写在同一行的块级公式（`$$x^2$$`）显示异常
- 修复在代码块处停留或选中文字时页面跳动
- 修复设置中的正文字号与标题字号对富文本模式不生效
- 修复保持模式下多列筛选时，筛选候选值未随其他列的筛选结果更新（现与 Excel 行为一致）
- 修复部分已转义图片路径在保持模式下被二次编码，导致相对图片无法正常显示的问题
- 修复 Ctrl/Cmd+0 在富文本编辑区内与段落快捷键冲突的问题：富文本中用于段落化，其他位置仍用于重置缩放
