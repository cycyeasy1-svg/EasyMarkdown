# EasyMarkdown — 一个窗口，装下所有 .md 文件

> 免费 · 开源 · 不要账号

一个免费的 Typora 平替，但不止于此。

- **下载 Windows 版**（.exe，NSIS）/ **下载 macOS 版**（.dmg，Apple Silicon & Intel）：https://github.com/cycyeasy1-svg/EasyMarkdown/releases/latest
- 构建未签名 — Windows：更多信息 → 仍要运行 · macOS：右键 → 打开

## 它能做什么

| # | 功能 | 说明 | 快捷键 |
| --- | --- | --- | --- |
| 01 | 标签页 | 双击一个文件，是多一个标签，不是多一个窗口。 | Ctrl/⌘ Tab |
| 02 | 文件夹工作区 | 整个文件夹挂在侧边栏，新建、重命名、删除都不用切出去。 | Ctrl/⌘ B |
| 03 | 所见即所得 | 打字就渲染。表格、代码高亮、LaTeX、任务清单都认。 | / |
| 04 | 富文本复制 | 复制自带格式，粘进微信公众号、邮件、Notion 都不丢样式。 | Ctrl/⌘ C |

更多：命令面板（Ctrl+P）、大纲面板、源码模式、PDF 导出、会话恢复、外部修改自动重载、中英双语界面。

## 六套主题

明亮、暗夜，以及四套莫兰迪：雾、鼠尾草、玫瑰、暮色。
